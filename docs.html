Requirements

  • Windows, Macosx, Linux
  • Node.js v12 or higher

Install

mac (Homebrew)

$ brew install Kennchan-cli

mac (NPM)

$ npm install Kennchan-cli

mac (Yarn)

$ yarn install Kennchan-cli
